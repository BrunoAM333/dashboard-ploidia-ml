03 Análise: Teste vs. Validação

Leitura principal

O maior ROC-AUC de teste observado foi 69.4% com EfficientNet-B0 + TabPFN Config 027, usando trofectoderme + mci + blastocele.

Como controle metodológico, o melhor modelo pela validação foi EfficientNet-B0 + TabPFN Config 027, usando trofectoderme + blastocele, com ROC-AUC de validação 69.3% e ROC-AUC de teste 68.9%.

A concordância entre os dois rankings sugere boa estabilidade dos modelos baseados em EfficientNet-B0.
