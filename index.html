Equilíbrio de Sensibilidades

euploide vs. aneuploide

Uma AUC alta não significa que o modelo está equilibrado. O mais importante clinicamente é identificar corretamente os euploides. Cada linha mostra quanto cada uma das três melhores redes acerta euploides (azul) e aneuploides (violeta) — e o Swin-Base é o único que avança além do cluster.

Cada linha = 1 rede · tamanho do dot = relativo à AUC
Sens. Euploide Sens. Aneuploide
As duas variantes Swin-Tiny têm sensibilidade euploide na faixa 0.397–0.410 (cluster esquerdo). O Swin-Base desloca o ponto azul para 0.497 — um ganho de +8,7 p.p. na detecção de euploides — enquanto mantém a sensibilidade aneuploide próxima dos demais (0.858 vs. 0.890–0.896).

Reprodução

Comandos para rodar cada experimento do zero. Todos usam semente fixa (SEED=42) e a mesma divisão de dados.

Treino Imagem Completa (Swin-Tiny) train_modality.py
python train_modality.py --modality completas --model swin_tiny \ --img_size 384 --batch_size 10 --epochs 50 \ --lr 5e-5 --weight_decay 0.1 \ --label_smooth 0.15 --drop_path_rate 0.4 \ --crop --cache
Fusão (Swin-Tiny) — antigo recorde fusion_complete_bl_te.py
python fusion_complete_bl_te.py \ --data_root ../../data/raw/images \ --lr 1e-3 --epochs 30
Treino BL com Swin-Base train_modality.py
python train_modality.py --modality bl --model swin_base \ --img_size 224 --batch_size 16 --epochs 50 \ --lr 3e-5 --weight_decay 0.05 \ --label_smooth 0.15 --drop_path_rate 0.3 \ --crop --cache
Fusão Campeã · COMPLETA+BL+TF (Swin-Base) ⭐ fusion_complete_bl_te_swinbase.py
